# Reactime Time Tester

A website designed to easily test your reaction time and to allow you to track your statistics, while keeping all of the data on your device.

## Features
- Precise reaction time measurement using performance.now()

- Randomized delay from 3-4 seconds

- Performance data saved on device (localStorage)

- Line chart visualization (Chart.js)

- Average + best score tracking

- Percentile calculation using Human Benchmark data

- Mobile-friendly

- No backend, no data collection

## How to use

- Click the Start button

- Wait for the screen to turn green

- Click as fast as possible

- View your reaction time and overall statistics below

## Tech Stack
I used a straightforward tech stack for this project, as it did not require any specific features from frameworks.

- HTML

- CSS

- Vanilla JavaScript

- Chart.js

- GitHub Pages (for deployment)
